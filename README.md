# FraudBusters Hackathon Project

This repository contains our Hackathon Spring 2026 project on FinTech fraud detection.

## Team
- Adeel Shahzad
- Riya Jain

## Project Summary
We built a layered AI/ML fraud detection framework using:
- Rules layer
- XGBoost
- Isolation Forest anomaly detection
- Graph-based features

## Final Results
- Precision: 28.65%
- Recall: 31.98%
- F1-score: 30.22%
